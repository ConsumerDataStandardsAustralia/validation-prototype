LambdaBank: CDR Sandbox

Sandbox (Mock Bank)

localhost:8000 is a hosted sandbox produced by QFPL to assist third-parties test CDR apps. The sandbox implements all of the defined resource endpoints in the Banking API, responding with mock data that is conformant to the CDR draft standards (v0.2.0). The latest version of the standards can be found at Consumer Data Standards Australia. The source code for this project can be found in the Conformance github repository.

For further information, please contact cdr@qfpl.io.
